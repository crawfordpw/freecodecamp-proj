Timestamp Microservice

  1. Can use a Unix timestamp in milliseconds
  2. Use a valid JavaScript date_string using newDate(date_string)
    • See JavaScript Date Object for more details.
    • Can use a valid DateTime string following the Date Time String Format in the ECMAScript specification.
    • Namely: YYYY-MM-DDTHH:mm:ss.sssZ

Use the endpoint [project_url]/api/timestamp/:date to see the microservice in action
